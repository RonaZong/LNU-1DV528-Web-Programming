About the course literature.

Online resources are available and detailed out in each part/lesson of the course and published on the course website.



### MDN Mozilla Developers Network

During the course we will use The Mozilla Developers Network as a guide and reference. You can see an overview of the content at "[Web technology for developers](https://developer.mozilla.org/en-US/docs/Web)".



## JavaScript for impatient programmers

<p><img src="https://exploringjs.com/impatient-js/img-homepage/cover-homepage.jpg" width="200px" alt="" /></p>

The author Dr. Axel Rauschmayer has a [set of JavaScript online books](https://exploringjs.com/) that are most valuable for anyone learning JavaScript.

My recommendation to a programmer who knows how to program is to start with the book "[JavaScript for impatient programmers](https://exploringjs.com/impatient-js/index.html)".



## Eloquent JavaScript

<p><img src="http://eloquentjavascript.net/img/cover.jpg" width="200px" alt="" /></p>

The free online book "[Eloquent JavaScript](https://eloquentjavascript.net/)" provides in insight into programming in general and JavaScript in particular.

Eloquent JavaScript, 3rd Edition by Marijn Haverbake, ISBN-13 978-1593279509.
