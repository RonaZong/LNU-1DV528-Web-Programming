This is the course website for "1DV525 – Introduction to web programming" at Linnaeus University, Sweden.



About
--------------------------------

This course will teach you the basics in modern web development, starting with an introduction to html5, css3 and javascript for programmers mostly familiar to java. After the introduction a deeper insight into client-side javascript is on the agenda.

On this website you will find all information about courses. All learning materials are open to anyone to take part in. If you are a course participant, you may need to log in to other systems to gain access to specific reviewers, etc., when these services are available in connection with the course start.
